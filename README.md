# Personal Information Manager

## Project Description
This is my first Python project.
The program stores personal information and asks the user for additional details,
then displays everything in a formatted output.

## What I Learned
- Variables and data types
- User input/output
- String formatting using f-strings
- Input validation
- Writing functions
- Basic program structure

## How to Run
1. Install Python
2. Open terminal in project folder
3. Run:

python personal_info.py

4. Enter the requested details

## Features
- Stores static information (name, age, city, hobby)
- Gets user input (favorite food, color)
- Validates empty input
- Calculates age in months
- Clean formatted output
